Talently — detail-page screenshots
==================================

Drop image files in THIS folder and they appear on /work/talently automatically
(no code changes needed). Files are shown in filename order, so prefix with
numbers to control order:

  01-overview.png   <- FIRST image = the big main/hero image at the top
  02-recruiting.png <- shown in the "Screens" gallery below
  03-pipeline.png
  ...

Supported: .png .jpg .jpeg .webp .avif .gif
Recommended width: ~1600px+ (the dashboard screenshots).

Save the dark "Overview" dashboard screenshot as 01-overview.png to make it the
main image. The card preview (before clicking) keeps using the live prototype.
